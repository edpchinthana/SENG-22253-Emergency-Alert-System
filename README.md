# SENG-22253-Emergency-Alert-System
Arduino based motorcycle emergency alert system
## Components
- Arduino Uno R3
- S-322 Shock sensor
- GY-521 Gyroscope module
- A7 GPS GPRS GSM module
- 16x2 LCD display
- 10k ohms resisters
- 10K potentiometer
- Active buzzer
- Push button
